# 命令与按键

我是 Alyce。*写快捷键说明页总觉得有点不好意思…但说实话，这种内容就是你半夜两点想不起怎么打开设置时最需要的东西。*

这里列的都是当前运行时里确确实实接好线的。没有"理论上应该有"的键，也没有"将来会有的"占位。

## Slash 命令

在主输入框里打，以 `/` 开头，回车立即执行。

### 基础命令

| 命令 | 干什么的 |
|---|---|
| `/help` | 列出全部可用命令。*迷路了就从这开始。* |
| `/settings` | 直接打开设置弹窗。 |
| `/setup` | 首次配置引导。 |
| `/clear` | 清空当前对话，重新开始。 |
| `/rewind` | 打开回退选择器，恢复到更早的一次输入前。 |
| `/exit` | 关掉 Alyce-Agent。 |

### 记忆

| 命令 | 干什么的 |
|---|---|
| `/remember <文字>` | 存进持久记忆——关了程序重开还在。 |
| `/remember --session <文字>` | 存进会话记忆——只活在这一轮会话里。 |
| `/memory` | 看看现在都记了些什么。 |
| `/memory clear` | 只清会话记忆。 |
| `/memory clear --all` | 全清——会话记忆和持久记忆都清。*慎用。* |

*持久记忆和会话记忆的区别很简单：想重启后还留着，就别加 `--session`；只是这次对话临时用的，就加上。*

### 上下文与模型

| 命令 | 干什么的 |
|---|---|
| `/context` | 预览模型下一轮实际会收到的完整 payload。*这东西调试起来特别有用——能让你看到记忆注入、压缩摘要等聊天记录里看不见的东西。* |
| `/context <文字>` | 跟上面一样，但额外加一段文字塞进上下文里。 |
| `/model <名称>` | 现场切换模型。比如 `/model gpt-4o`。 |

### 目录范围

| 命令 | 干什么的 |
|---|---|
| `/add-dir <路径>` | 临时把某个目录加到助手的文件访问白名单里。 |
| `/add-dir --save <路径>` | 同上，但跨会话保留。 |

*默认情况下 Alyce-Agent 只能碰工作区内的文件。用 `/add-dir` 让它够到外面——比如另一个项目里的共享库。*

### 会话记录

| 命令 | 干什么的 |
|---|---|
| `/resume` | 打开历史会话选择器，挑一个以前的对话接续。 |
| `/resume <id 或关键词>` | 按 ID 或关键词直接恢复指定会话。 |
| `/sessions` | 列出项目里最近保存的会话。 |

*我几乎天天用 `/resume`。晚上关终端，第二天早上起来，一个命令就回到昨天停下的地方。*

## 全局快捷键

这些在应用里任何地方都有效——不管什么弹窗开着。

| 按键 | 效果 |
|---|---|
| `Ctrl+Q` | 退出程序。*没有确认弹窗，所以想好了再按。* |
| `Ctrl+X` | 打开设置。*大概除了打字之外用得最多的键。* |
| `Ctrl+O` | 打开当前选中消息的详情。看长工具输出时特别管用。 |
| `Esc` | 请求运行时中断 Alyce；空输入时打开回退；在回退选择器里连续按会往更早的输入移动。 |

## 中断

| 按键 | 效果 |
|---|---|
| `Ctrl+C` | 清空你正在输入的内容。如果模型请求正在跑，就中断那个请求。 |

某轮被打断后，空输入时按 `Esc` 可以选择回退到哪一次输入前。选中后按 `Enter`；如果有 Alyce 跟踪到的文件编辑快照，也可以选择同时恢复代码和对话。

## 对话导航

| 按键 | 效果 |
|---|---|
| `Up` | 移到上一条消息。 |
| `Down` | 移到下一条。 |

*看着简单，但翻看助手五轮前干了什么的时候，你会反复按这两个键。*

## 滚动

| 按键 / 动作 | 效果 |
|---|---|
| 鼠标滚轮上/下 | 滚动对话区。 |
| `PageUp` / `PageDown` | 整页翻。 |
| `Home` / `End` | 跳到当前视图的顶部或底部。 |
| `Ctrl+Home` / `Ctrl+End` | 跳到对话的最开头或最末尾。 |
| `Ctrl+0` | 重置滚动位置。 |

## 设置弹窗操作

按 `Ctrl+X` 后会弹出设置界面。操作方式如下：

### 通用

| 按键 | 效果 |
|---|---|
| `Left` / `Right` | 在连接和会话两个标签页间切。 |
| `Up` / `Down` | 在字段间上下移动。 |
| `Enter` | 编辑当前字段、或翻转布尔开关、或循环选项。 |
| `S` | 保存所有改动。 |
| `Esc` | 不保存直接关。 |

### 连接标签页专属

| 按键 | 效果 |
|---|---|
| `P` | 在**项目级**和**用户级**之间切换保存目标。项目配置存 `./.alyce/config.json`，用户配置存 `~/.alyce/config.json`。 |

*保存范围的开关挺容易忽略的。如果你把 API key 存成项目级然后又分享了仓库…嗯，别这么干。*

## 一些可能让你意外的细节

- **文本字段**把 `\n` 当真正的换行符。想在 prompt 字段里换行，就打 `\n`。
- **数字字段**会自动规整成正整数。输入 `-5` 或 `3.7` 会被纠正。
- **开关字段**显示 `on` 或 `off`。按 `Enter` 翻转。
